# Frontend Gulp Build System

A modern frontend build setup using **Gulp 5** with support for SCSS, HTML, image optimization, live reload, and production builds.

## Clone the repository

```bash
git clone https://github.com/anovacodes/gulp.git
cd gulp
```

## Install dependencies

```bash
npm i
```

## Run in development mode

```bash
npm run dev
```

## Open project in your browser

You can open the project in the browser after it's been started using [this link](http://localhost:3000/).

## Build for production

```bash
npm run build
```

Generates optimized files into the **dist** folder:

- Minified HTML, CSS, JS
- Optimized images
- Auto-prefixer for CSS

## Requirements

- Node.js: v22.12.0 (as specified in package.json)
